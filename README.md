# Diwali Sales Analysis

##  Project Overview

This project analyzes Diwali sales data using Python to understand customer purchasing behavior and identify business insights. The analysis focuses on different customer demographics such as gender, age group, marital status, occupation, state, and product category to discover patterns that can help businesses improve their marketing strategies and increase sales.

The project includes data cleaning, exploratory data analysis (EDA), and data visualization using various Python libraries.

---

##  Dataset

The dataset contains customer purchase information during the Diwali festival.

Some of the important columns include:

- User_ID
- Gender
- Age
- Age Group
- State
- Marital Status
- Occupation
- Product Category
- Orders
- Amount

##  Key Insights

After analyzing the dataset, the following observations were made:

- Female customers made higher purchases compared to male customers.
- Customers between the age group of **26–35 years** contributed the highest sales.
- Married women showed a higher purchasing trend.
- States like **Uttar Pradesh, Maharashtra, and Karnataka** generated the highest sales.
- Customers working in **IT, Healthcare, and Aviation** sectors spent more during Diwali.
- Certain product categories contributed significantly more to total revenue.
- A small number of products accounted for a large share of total orders.
